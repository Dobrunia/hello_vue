Кейсы для изучения Vue композаблов
1. Поиск и фильтрация задач
ref — поисковый запрос и фильтр (all/done/undone)
computed — отфильтрованный список задач
2. Статистика на главной странице
computed — статистика (всего, выполнено, осталось, прогресс)
watch — логирование изменений статистики
3. Форма добавления задачи
reactive — форма (title, done)
computed — валидность формы
watch — валидация в реальном времени
4. Автосохранение с задержкой
ref — таймер для debounce
watch — отслеживание изменений задач с задержкой
5. Редактирование задачи на лету
ref — флаг режима редактирования
reactive — форма редактирования
6. Счетчик выполненных задач
computed — количество выполненных
watch — реакция на изменения (например, показ уведомления)
7. Локальное хранилище (localStorage)
ref — данные для сохранения
watch — автосохранение в localStorage при изменениях
8. Таймер обратного отсчета
ref — оставшееся время
watch — обновление каждую секунду
9. Модальное окно
ref — видимость модального окна
watch — блокировка скролла при открытии
10. Сортировка и пагинация
ref — текущая страница, тип сортировки
computed — отсортированный и разбитый на страницы список
